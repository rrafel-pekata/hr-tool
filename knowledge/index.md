# Índice de Base de Conocimiento — Pekata ATS

Este archivo describe el contenido de cada archivo de la base de conocimiento.
Se usa para determinar qué archivos consultar según la pregunta del usuario.

## Archivos disponibles

### primeros-pasos.md
Qué es Pekata ATS, para quién es, flujo general de uso, primer login, primeros pasos tras crear cuenta, conceptos clave del sistema.

### empresas.md
Crear y editar empresa (tenant), campos disponibles, mejorar descripción con IA, dashboard de empresa con KPIs, gestión multi-tenant, logo y datos corporativos.

### posiciones.md
Crear posiciones/ofertas de empleo, generar descripción con IA, estados de posición (borrador, publicada, pausada, cerrada), filtros, tipos de contrato, campos disponibles.

### candidatos.md
Agregar candidatos a una posición, subir CV (PDF), análisis automático de CV con IA, pipeline de candidatos, estados de candidatura, puntuación de encaje, notas, estrellas/rating.

### entrevistas.md
Programar entrevistas para candidatos, tipos de entrevista, registrar resultado, puntuación, editar entrevistas.

### casos-practicos.md
Crear caso práctico manual o con IA para una posición, enviar caso a candidato por email, portal de entrega del candidato, evaluar entregas.

### evaluacion-ia.md
Evaluación integral de candidato con IA, qué datos se analizan (CV, entrevistas, caso práctico), puntuaciones, regenerar evaluación.

### roles-y-permisos.md
Tipos de usuario (Admin y Recruiter), qué puede hacer cada rol, gestión de usuarios.

### planes.md
Información sobre planes y funcionalidades (actualmente todas disponibles en plan gratuito), futuras limitaciones.

### navegacion.md
Estructura del sidebar, URLs principales de la aplicación, cómo navegar entre secciones, tours de ayuda, atajos.
